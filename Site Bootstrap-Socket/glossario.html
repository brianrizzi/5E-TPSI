Glossario

A.

ARP

Il protocollo ARP (Address Resolution Protocol) è utilizzato per mappare un indirizzo IP a un indirizzo MAC in una rete locale.

ASCII

ASCII (American Standard Code for Information Interchange) è un sistema di codifica dei caratteri che rappresenta testo e simboli usando numeri.

E.

Ethernet

Ethernet è una tecnologia di rete utilizzata per collegare dispositivi in una rete locale (LAN). Utilizza cavi fisici (come il cavo in rame o fibra ottica) per trasmettere dati tra dispositivi.

F.

FTP

FTP (File Transfer Protocol) è un protocollo di rete utilizzato per trasferire file tra un client e un server su una rete, come Internet.

H.

Handshake

L'handshake è un processo di negoziazione iniziale in una comunicazione di rete, in cui due dispositivi si scambiano informazioni per stabilire una connessione sicura e definire parametri di comunicazione, come protocolli e chiavi di crittografia.

HTTP

HTTP (HyperText Transfer Protocol) è un protocollo di comunicazione utilizzato per trasferire dati sul web.

I.

ICMP

ICMP (Internet Control Message Protocol): È un protocollo di rete utilizzato per inviare messaggi di controllo e di errore tra dispositivi, come nei comandi "ping" per testare la connettività di rete.

ISO/OSI

ISO/OSI (International Organization for Standardization/Open Systems Interconnection): È un modello di riferimento che descrive come i dati vengono trasmessi in una rete. È composto da 7 livelli, ciascuno responsabile di una parte del processo di comunicazione, dalla trasmissione fisica dei dati fino alla presentazione delle informazioni.

IP

IP (Internet Protocol): È un protocollo di rete che gestisce l'indirizzamento e il routing dei dati tra dispositivi su una rete, utilizzando indirizzi IP univoci per identificare i dispositivi.

J.

JPEG

JPEG (Joint Photographic Experts Group) è un formato di compressione delle immagini che riduce le dimensioni dei file mantenendo una qualità visiva elevata.

M.

MAC

MAC (Media Access Control) è un identificatore univoco assegnato a ogni dispositivo di rete, come una scheda di rete, per permettere la comunicazione all'interno di una rete locale (LAN).

N.

NetBIOS

Il NetBIOS è un protocollo di rete che permette la comunicazione tra computer su una rete locale (LAN), fornendo servizi come nomi, sessioni e datagrammi per identificare dispositivi e condividere risorse.

P.

Processi

I processi sono programmi in esecuzione, costituiti da codice, dati e risorse necessarie per funzionare.

PPP

Il PPP (Point-to-Point Protocol) è un protocollo per trasmettere dati tra due nodi di rete, spesso usato per connessioni internet.

S.

Server

Un server è un computer o software che fornisce servizi o risorse ad altri dispositivi (client) su una rete.

SMB

(Symmetric Multiprocessing) Architettura in cui più processori condividono memoria e risorse per eseguire compiti simultaneamente.

SMTP

(Simple Mail Transfer Protocol) Protocollo per inviare email su reti.

Software

Insieme di programmi e istruzioni che fanno funzionare un computer.

SSL

(Secure Sockets Layer) Protocollo per cifrare comunicazioni su internet.

T.

TCP

(Transmission Control Protocol) Protocollo che garantisce trasmissioni di dati affidabili tra dispositivi.

TLS

(Transport Layer Security) Protocollo per criptare e proteggere le comunicazioni su internet.

TCP/IP

Insieme di protocolli (TCP e IP) per comunicare e instradare dati su reti come internet.

U.

UDP

(User Datagram Protocol) Protocollo per trasmettere dati rapidamente, senza verifica.

USB

(Universal Serial Bus) Standard per collegare e alimentare dispositivi.

W.

Wi-Fi

Tecnologia wireless per connettere dispositivi a una rete.
